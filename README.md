## 📒 프로젝트 소개
분배법칙은 분리배출 정보를 얻고 분리배출에 직접 참여할 수 있는 서비스입니다.<br/>
기능은 다음과 같습니다. <br/>
1. 검색을 통해 분리배출 정보를 얻을 수 있습니다.
2. 분리배출을 인증하여 포인트를 받고 나무를 키울 수 있습니다.
3. 친환경 펀딩을 등록, 후원하여 환경보호를 실천할 수 있습니다.
<br/><br/>
